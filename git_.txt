
'================================= Git ==================================='

git - это программное обеспечение, для контроля версии

gir init - идёт инициализация нашего обьекта и создаётся папка .git (в ней находятся все необходимые файлы нашего репозитория)

git add (file or .) - добавление файлов в рабочие папке в индекс(Индекс в Git - это специальная промежуточная область в которой хранятся изменение файлов на пути от рабочей директории до репозитория)

git commit -m 'комментарий' - добавляет все файлы которые находятся в индексе во врутреннюю базу данных и сохраняет их состояние на даннвц момент

'git remote add {url}'
git remote add origin (ссылка ssh или http) - добавляет удаленные репозитории ккоторые находятся на каком нибудь сервере

'git push {url} {branch}'
git push origin master - отправляет наши изменения в удаленный репозиторий

'git pull {url} {branch}'
git pull origin master - стягивает изменения из удаленного репозитория

git branch - показывает список веток

'git branch {name branch}'
git branch dev - создаёт ветку на локальной репозитории

'git checkout {name branch}'
git checkout dev - переключение на ветку dev 




----------------------------------------------------------------------

1. создаём репозитории 
2. создаём локальный репозитории - git init
3. добавляем наши изменение в index - git add {. or file}
4. добавляем наши изменения из index в бд(локал репозитории) - git commit -m 'message'
5. связываем локальный репозитории с удаленным - git remote add origin {ссылка на удаленный репозитории SSH or HTTPS}
6. отправляем наши изменение из локального репозитора в удаленный репозиторий - git push origin master


git checkout hash_commit -> преключение к указанному комиту
git remote -v -> просмотр списка удаленных репозиториев
git remote rm/remove origin/<name> -> удаление ссылки на удаленный репозитории
git clone <ssh/http> -> скопировать склонить/скачать/содержимое удаленного репозитория (репо github)
